Configuration steps for Microsoft Teams

Go to Microsoft Azure AD and Login to your Microsoft account and go to Active Directory

In Acive Directory click on App regestration > New Registration

Fill out the name for your application and paste the redirected url which will be generated in OpenEduCat Settings Create

Copy and save the Application ID which will be used further

Now click on Certificates and Secrets, then create and save client secret key.

To add permissions click on API Permissions > Add Permissions > Delegated Permissions.

Search and add OnlineMeetings.Read & OnlineMeetings.ReadWrite then click on register.

Microsoft Teams Integration in OpenEduCat

Company Level Configuration

Go to Menu Settings --> OpenEducat. Here you can Configure your API Key,API Secret.

Once you have entered your API Key and API Secret, click on Authenticate button to generate access token.

Here, you can also select whether to send the cards of scheduled meetings as a notification for general channel or private channels

User Level Configuration

Now, Go to menu Timetables -> Sessions. Here Select Particular Session on which you want to create meeting.

Click on button online meeting the calender will open with all session details

Click on button Create teams meeting. the wizard will open .In that wizard, Here Invite via Email boolean, the email will be sent to all the attendees of the session for this video conference meeting

Here under the Invitation Tab, you can see the url of all the attendees of that session.

The attendees will receive a link to join with the invitation mail.
